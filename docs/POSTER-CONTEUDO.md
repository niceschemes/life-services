## Resumo

O Life Services é uma aplicação web desenvolvida para auxiliar pequenas empresas e prestadores de serviço na organização de clientes, estoque, orçamentos, ordens de serviço e controle financeiro básico. O projeto foi construído com Node.js, Express, MongoDB, HTML, CSS e JavaScript, com foco em demonstrar conceitos de computação em nuvem, autenticação, deploy, qualidade de software e aplicação prática em um contexto extensionista.

## Desafio

Pequenas empresas frequentemente utilizam controles manuais, planilhas ou mensagens para registrar clientes, peças, orçamentos e serviços. Essa prática pode gerar perda de informações, retrabalho, falta de histórico e dificuldade de acompanhamento financeiro.

## Síntese das ações

- Levantamento de necessidades de uma pequena empresa/prestador de serviço.
- Desenvolvimento de uma aplicação web funcional.
- Implementação de login e autenticação.
- Cadastro e gestão de clientes.
- Controle de estoque com entrada e saída.
- Geração e controle de orçamentos.
- Registro de ordens de serviço.
- Controle financeiro básico.
- Preparação da aplicação para deploy em nuvem.
- Organização do código conforme práticas de qualidade.

## Resultados

O projeto resultou em um sistema web funcional, capaz de centralizar informações operacionais importantes. A aplicação contribui para maior organização, controle de estoque, formalização de orçamentos e registro de serviços prestados, além de demonstrar a aplicação prática dos conceitos de computação em nuvem e qualidade de software.
