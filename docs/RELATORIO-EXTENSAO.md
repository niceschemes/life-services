## Descrição da atividade

Desenvolvimento de uma aplicação web chamada Life Services, voltada para auxiliar uma pequena empresa ou prestador de serviço no controle de clientes, estoque, orçamentos, ordens de serviço e movimentações financeiras básicas.

## Contextualização

O projeto foi desenvolvido a partir da identificação de dificuldades comuns em pequenas empresas, como controle manual de estoque, ausência de histórico organizado de clientes, dificuldade na emissão de orçamentos e falta de centralização das ordens de serviço.

## Desafios

- Definir um escopo simples e funcional.
- Organizar dados de clientes, produtos e serviços.
- Criar uma aplicação web acessível pelo navegador.
- Preparar o sistema para hospedagem em ambiente de nuvem.
- Aplicar práticas básicas de qualidade de software.

## Síntese das ações

Foi criada uma aplicação web com login administrativo, cadastro de clientes, controle de estoque com entrada e saída, geração de orçamentos, registro de ordens de serviço e controle financeiro básico. A aplicação utiliza Node.js, Express e MongoDB, com estrutura preparada para deploy em nuvem.

## Aspectos positivos

- Centralização das informações.
- Redução de controles manuais.
- Facilidade para gerar orçamentos.
- Controle simples de estoque.
- Aplicação prática de computação em nuvem.
- Organização do código e documentação.

## Dificuldades encontradas

- Definição do escopo adequado para o prazo.
- Organização entre funcionalidades essenciais e recursos avançados.
- Preparação da aplicação para ambiente de nuvem.
- Adequação do projeto aos critérios acadêmicos.

## Resultados atingidos

Foi obtido um sistema web funcional, capaz de demonstrar na prática os conceitos de computação em nuvem, desenvolvimento web, banco de dados, autenticação e qualidade de software. O projeto oferece uma solução inicial para controle operacional de pequenas empresas.

## Sugestões e observações

Como evolução futura, o sistema pode receber integração real com pagamentos, emissão fiscal, envio de orçamentos por WhatsApp/e-mail, relatórios avançados e painel mobile.
